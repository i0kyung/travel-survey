# travel-survey

여행을 떠나기 전 필요한 정보를 설문 형식으로 수집하고,  
사용자의 여행 성향과 조건을 바탕으로 여행 계획에 활용할 수 있는 결과지를 만들어주는 웹입니다.

## 주요 기능

- 여행지, 출발지, 날짜, 기간 입력
- 동행자 유형 선택
- 여행 분위기 선택
- 예산, 이동 방식, 숙소 선호 입력
- 음식 취향, 알레르기, 피하고 싶은 요소 정리
- 여행자 결과 리포트 생성
- 결과 복사 및 링크 공유 기능

## 목적

여행 계획을 바로 세우기 전에  
여행자의 취향, 예산, 동행자, 제약 조건을 먼저 정리하기 위한  
여행 정보 수집 웹입니다.
